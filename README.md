# topic-modeling-article-pair-similarity
Dataset Similarity Artikel Topic Modeling. 

Data artikel diperoleh dari Google Scholar menggunakan Publish or Perish. Query yang digunakan "Topic modeling" dengan batas waktu publikasi sejak 2019. Keyword diisi dengan Mendeley auto update dan sisanya dicari secara manual. Data berupa buku dan data tanpa keyword dibuang. Data dipasangkan secara kombinatorik dari 50 data menjadi 1225.
